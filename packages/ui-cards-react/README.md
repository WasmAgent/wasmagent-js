# @agentkit-js/ui-cards-react

React components — `MarkdownCard`, `D2Card`, `CardRenderer`, `ChatMessage`.

> Part of [agentkit-js](https://github.com/telleroutlook/agentkit-js) — a TypeScript + WASM agent runtime.

## Install

```bash
npm install @agentkit-js/ui-cards-react @agentkit-js/ui-cards react react-dom
```

## Usage

```tsx
import { ChatMessage } from "@agentkit-js/ui-cards-react";
```
